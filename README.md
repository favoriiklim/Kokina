Kokina Widget
This is a small desktop application developed during my free time for personal project purposes. It is a minimalist Kokina (Christmas Berry) widget that sits above the system clock area and features a growth animation upon startup.

Features
Smooth Startup Animation: Branches, leaves, and berries appear in sequence.
Minimalist Design: Transparent background and borderless window design.
Mouse Interaction: Left-click restarts the animation, right-click exits the application.

Technical Details
Developed using .NET 8.0 and WPF (Windows Presentation Foundation).
Graphics are created using XAML Path geometries and 3D-effect gradient brushes.
Can be published as a single executable (Self-contained).

Installation:
Clone the repository to your computer.
Open the Kokina.sln file with Visual Studio 2022.
Build and run the project in Release mode.
